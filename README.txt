Introduction
============
Verkkomaksut provide payment processor which covers most of Finnish digital payments.
"getpaid.verkkomaksut" implements Verkkomaksut payment processors to gaipaid.

Intallation
===========
To use getpaid.verkkomaksut describe those below to buildout.cfg

    eggs =
        getpaid.verkkomaksut
    
    zcml =
        getpaid.verkkomaksut
